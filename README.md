# 桌面电子宠物 (Electronic Pet)

一个复刻经典QQ宠物体验的桌面养成游戏。

## ✨ 核心特性

*   **桌面陪伴**: 宠物会在你的桌面上自由漫步、休息、玩耍，拥有可爱的动画和表情。
*   **属性养成**: 照顾宠物的饥饿度、清洁度、心情和健康。
*   **经济系统**: 
    *   **打工**: 派遣宠物去打工赚取元宝（服务员、建筑工、程序员）。
    *   **学习**: 提升各项能力（武力、智力、魅力）以解锁更高薪的工作。
    *   **商店**: 购买食物、清洁用品、玩具和药品。
*   **互动系统**:
    *   抚摸互动。
    *   右键菜单快捷操作。
    *   丰富的对话气泡。
*   **成长进化**: 从蛋开始孵化，经历幼年期，最终进化为成年期。
*   **探险系统**: 派遣宠物去不同地点探险，获取随机奖励。
*   **贴心功能**:
    *   **置顶显示**: 保持宠物始终在窗口最上层。
    *   **边缘停靠**: 拖拽至屏幕边缘自动隐藏，鼠标悬停时探出。
    *   **托盘最小化**: 一键隐藏到系统托盘，不打扰工作。
    *   **多语言**: 支持中文和英文。

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发模式

```bash
npm start
```

### 打包构建

```bash
npm run build
npm run make
```

## 🎮 操作指南

*   **左键拖拽**: 移动宠物位置。
*   **左键点击**: 抚摸宠物，增加心情。
*   **右键点击**: 打开功能菜单。
    *   **状态**: 查看宠物详细属性。
    *   **喂食/洗澡/玩耍**: 快速照顾宠物。
    *   **打工/学习/商店/背包**: 打开对应功能窗口。
    *   **探险**: 开启随机探险事件。
    *   **隐藏到托盘**: 暂时隐藏宠物。
    *   **置顶显示**: 切换窗口置顶状态。

## 🔧 技术栈

*   **Electron**: 跨平台桌面应用框架
*   **TypeScript**: 强类型语言保证代码质量
*   **Canvas API**: 高性能像素画渲染

## 📂 数据存储

游戏存档保存在本地用户目录：
`%AppData%/ElectronicPet/save.json` (Windows)
